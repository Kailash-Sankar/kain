# kain
 Koa boilerplate with essentials
 
 Middlewares, Routing and a health check api

## contains
 * helmet - security headers
 * CORS
 * body parser
 * access logs
 * routing
 
## setup
  clone repo
  
  `yarn install`
  
  `yarn start`
  
   Runs with nodemon on dev
